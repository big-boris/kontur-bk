Отчётность в контролирующие органы
02

Отчетность в сервисе формируется автоматически на основе заведенных ранее бухгалтерских данных. .

Все возможности сервиса соответствуют законодательству, формы обновляются сами, а встроенная система проверки предупредит об ошибках при заполнении.

Отправляйте сформированные отчёты непосредственно из сервиса, подписав их электронной подписью.

Расчёт зарплаты, больничных, отпускных, налогов и взносов
03

Импортируйте из 1С данные по всем сотрудникам, а также справочники контрагентов, номенклатуры и основных средств.

Зарплата по каждому сотруднику будет автоматически начисляться каждый расчетный период — вам останется только подтвердить суммы. Сам расчёт всех выплат займёт несколько минут.

Бухгалтерский и налоговый учет для УСН, ЕНВД и ОСНО
04

В системе есть вся нормативная информация и документация для различных систем налогообложения: ОСНО, УСН, ЕНВД.

Выполняйте все необходимые операции прямо в системе:

  • анализ счетов, ведение карточек счетов и журнала проводок;
  • сверка с контрагентами по НДС;
  • книги покупок и продаж, журнал учета счетов-фактур;
  • книга учета доходов и расходов для УСН;
  • подсказки и напоминания об уплате налогов;
  • сверка с контролирующими органами, таблица задолженности по налогам;
  • учёт основных средств.
Информация, необходимая для ведения бизнеса
05

Сервис интуитивно понятен и прост. Если у вас возникают вопросы, можете воспользоваться многочисленными подсказками. Директор и менеджеры могут сами выставлять счета, акты, накладные, а входящие документы распознаются автоматически по сканам.

Электронный документооборот
06

Оформляйте и получайте электронные акты, счета-фактуры и накладные и отправлять их прямо в сервисе.

Накладные с большим количестовм данных можно распознать по сканам и фотографиям.

Передавайте платежные документы между Бухгалтерией и интернет-банком автоматически. Документы с подписью и печатью можно отправлять по почте прямо из Бухгалтерии.
